Método Espectral FFT Avanzado

Genera ruido 1/f mediante la construcción directa del espectro de potencia deseado S(f) ∝ 1/f^α, aplicando fase aleatoria y transformada inversa de Fourier. Incluye:

  • Ventanas espectrales: Reducen el leakage espectral
  • Filtros anti-aliasing: Eliminan frecuencias no deseadas
  • Suavizado adaptativo: Minimiza discontinuidades
  • Normalización robusta: Mantiene propiedades estadísticas
⚠️ Nota: Para aplicaciones críticas como el modelo MFSU (∂ ≈ 0.921), se recomienda usar el método Timmer-Koenig con suavizado Savitzky-Golay para minimizar artefactos.
